Welcome to your introductory git-class!

To go through the basics we will be using this test-repo that I created to get a look at github-pages. We will use it so you can try out basic operations and get an initial view on parts of the workflow that git uses.

We will:

  • get stuff from an online-repo, change things locally, and then put those changes back into the online-repo
  • make branches, bring different branches together
  • provoke some conflict, solve the conflict

Maybe we even learn how to customize git a little bit more to our needs!
